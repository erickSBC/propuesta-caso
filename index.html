Preguntas Técnicas Frecuentes

¿Cómo funciona el despliegue "Zero-Ops"?
Detectamos automáticamente el runtime de tu proyecto tras un git push. Basado en un clúster de Kubernetes altamente optimizado, abstraemos la complejidad para ofrecerte una experiencia Serverless total.
¿Qué medidas de seguridad están implementadas?
Aislamiento mediante VPC, Firewalls de aplicación (WAF) y mitigación DDoS de hasta 100Gbps. Cumplimos con ISO 27001 y SOC 2 Type II para garantizar la máxima seguridad de tus datos.
¿Cómo se factura el uso de recursos?
Utilizamos un modelo Pay-as-you-go facturado por segundo de uso real de CPU y RAM. Sin costos ocultos por transferencia de datos interna, optimizando tus costos operativos.
